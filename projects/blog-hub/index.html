Blog Hub

Blog hub is a dynamic web application that lets users to read and publish articles. The platform is an example of social journalism, having a hybrid collection of amateur and professional people and publications, or exclusive blogs or publishers and is regarded as a blog host.

Features included:

  • Medium-like editor for styling fonts
  • User authentication and authorisation
  • CRUD operations on the articles
  • Draft section
  • Clap (or like) posts and add comments
  • Author can customise comments on his blogposts
  • Simplified yet eye-pleasing UI

Framework: Django

Third parties used: Medium-editor
